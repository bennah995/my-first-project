user types in the search area in their browser and press enter. The browser converts the URL(what you typed) to get the domain name. It then asks the DNS(Domain Name Server), to translate that domain into an IP address.

With IP address, the device connects to the hosting server. Browser then sends HTTP request to that server, asking for the website's content. Server processes the request and sends back the web files. The browser readsreads and interprets the response for the user.


The internet is just connected computer networks, while the World Wide Web is a system of interlinked docs, services and websites that are accessed through computers
